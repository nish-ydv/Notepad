# Notepad - Modern Tkinter Notepad
A simple Notepad application built with
**python + Tkinter**.
Features dark/light mode, live word count and file management.

## Features
- Create, open, edit, save and delete text files
- Real-time word counter
- Dark and light mode toggle
- keyboard shortcuts: ctrl+o (open) and ctrl+s (save)
- user-friendly interface

## How To Run
```bash
python notepad.py
